📋 Project Brief — CampaignFlow Tracker
Built by Sun Cerrae Quinones · Digital PM Portfolio Project #1
v1.0 · Live
Problem
Manual campaign tracking across 10+ concurrent projects
Solution
Real-time Agile board with KPI dashboarding
Impact
~40% reduction in missed deadlines (modeled)
Context: Managing 10+ concurrent digital campaigns at Belk and Victoria's Secret with JIRA required significant context-switching and manual status updates. This app centralizes campaign visibility across four workflow stages, enabling stakeholder updates at a glance.

Stack: Vanilla HTML/CSS/JS · Drag-and-drop API · Local storage persistence · Responsive grid layout
Methodology: Agile/Scrum · Kanban board · Sprint-style campaign management
