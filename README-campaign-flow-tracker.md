# ⚡ CampaignFlow Tracker

> A real-time Agile Kanban board for managing concurrent digital marketing campaigns — built from experience coordinating 10+ campaigns simultaneously at Belk and Victoria's Secret & Co.

**[🚀 Live Demo](https://cerrae.github.io/campaign-flow-tracker)** &nbsp;|&nbsp; Built by [Sun Cerrae Quinones](https://linkedin.com/in/suncerrae)

---

## 📋 Project Brief

| | |
|---|---|
| **Problem** | Managing 10+ concurrent digital campaigns with manual tracking caused missed deadlines, stakeholder misalignment, and no single source of truth |
| **Solution** | A centralized Kanban board with live KPI dashboarding, drag-and-drop workflow, and priority filtering |
| **Impact** | Modeled ~40% reduction in missed deadlines based on workflow improvements implemented at Belk (Oct 2023) |
| **Methodology** | Agile/Scrum · Kanban · Sprint-based campaign management |

---

## ✨ Features

- **Kanban Board** — Drag-and-drop campaign cards across four workflow stages: To Do → In Progress → In Review → Done
- **Priority Filtering** — Filter by High / Medium / Low priority across all columns simultaneously
- **Live Search** — Instant search across all active campaigns
- **KPI Stats Bar** — Real-time count of campaigns at each stage
- **Campaign Velocity Chart** — 6-sprint bar chart tracking planned vs. completed campaigns
- **Activity Feed** — Live log of every status change and new campaign added
- **Add / Delete Campaigns** — Full CRUD via modal form with channel tagging, due dates, and assignees
- **Persistent Storage** — Campaign data saved to localStorage; survives page refresh

## 💼 Real-World Context

This project is directly informed by my work as a **Digital Producer at Belk** (Jun–Oct 2023), where I:

- Coordinated full-cycle digital production across **10+ concurrent campaigns** using JIRA and MS Project
- Translated business requirements into actionable tasks to improve team alignment and execution
- Led QA and publishing workflows, proactively identifying risks and resolving blockers to meet deadlines

CampaignFlow replicates the visibility layer I wished existed — a single board where stakeholders could see campaign status without a standup.
